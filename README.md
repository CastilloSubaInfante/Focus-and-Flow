# Focus & Flow

Focus & Flow is a Flutter-based productivity and habit-tracking application designed for students and young professionals. The app helps users manage their daily routines mmore efficiently by reducing procastination and encouraging consistent habits. Through smart reminders, progress tracking, and motivating messages, Focus & Flow aims to support users in staying organized, focused, and aligned with their personal goals.

# Features 
  - Add tasks, set reminders and receive notifications
  - Track task progress with a dashboard and monthly calendar view
  - Sign up, log in, log out, and verify accounts through email
  - Edit profile, change password, and manage user credentials
  - Dark mode, user-friendly interface, and smooth navigation
  - Works on mobile and Chrome (web version supported)

# How to Run
  - Clone the repository: git clone <repository-link>
  - Navigate to the project folder: cd focus_and_flow
  - Install dependecies: flutter pub get
  - Run the application: flutter run
